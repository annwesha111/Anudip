Overview :
This Power BI dashboard analyzes Blinkit's sales performance across outlet types, locations, and product categories. 
It provides insights into sales trends, customer preferences, and overall business performance.

Key Insights :
- Supermarket Type 1 generates the highest total sales.
- Tier 3 outlets contribute the most to overall sales.
- Fruits, Snacks, and Household items are top-performing categories.
- Low Fat products slightly outperform Regular products.
- Average rating remains consistent across outlets (~3.9).

Tools Used :
- Power BI
- Excel (Data Source)
- Data Cleaning & Transformation (Power Query)
- DAX (for calculations)

Features :
- Interactive dashboard with slicers (Outlet Location, Size, Item Type)
- KPI cards for Total Sales, Avg Sales, No. of Items, Avg Rating
- Visualizations:
  - Donut charts
  - Bar charts
  - Line chart
- Navigation buttons (Home & Reset Filters)
